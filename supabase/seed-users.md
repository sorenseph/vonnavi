# Usuarios admin - Von Navi

## Crear usuarios en Supabase

1. Ve a **Supabase Dashboard** → **Authentication** → **Users** → **Add user** → **Create new user**
2. Crea estos 2 usuarios:

| Email | Contraseña | Uso |
|-------|------------|-----|
| admin@vonnavi.com | Admin123 | Administrador (ver, editar todo) |
| ventas@vonnavi.com | Ventas123 | Ventas (ver, editar cotizaciones y reservaciones) |

3. Deja activo **Auto Confirm User** para acceso inmediato.
4. Inicia sesión en `/login` con cualquiera de estas credenciales.
